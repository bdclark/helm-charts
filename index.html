Helm Charts Repository

Usage

Add this Helm chart repository:

helm repo add bdclark https://bdclark.github.io/helm-charts/
helm repo update

Search Charts

helm search repo bdclark

Install a Chart

helm install my-release bdclark/<chart-name>

Available Charts

View the complete list of available charts and versions in the index.yaml file.
